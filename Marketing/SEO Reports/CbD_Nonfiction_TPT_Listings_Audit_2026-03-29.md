## The Universal Problem Across All 6 Units

Before unit-by-unit breakdowns: every nonfiction listing shares the same structural gap.

**The hooks are written for practitioners who already know CbD.** The descriptions open with instructional methodology ("teach CER through annotation," "ask what really happened?") rather than with the search terms buyers used to find the listing. TPT's description preview in search results shows approximately 130–180 characters — and right now, most of those previews contain zero product-specific keywords.

The fix is the same for all six: **keyword-first opening, then the hook.** You don't lose the compelling description — you shift it to sentences 2–3, after the algorithm has already matched the buyer to the product.

**The single biggest gap:** "Adapted reading" appears nowhere in any current title or description opening. This is the most searched long-tail phrase for this product type, with near-zero competition in the special education TPT niche. It needs to be in every title.

## Conversion Analysis — Beyond SEO

Getting found is half the battle. Getting the click-to-purchase is the other half. Here's what the CbD nonfiction line needs on the conversion side:

### 1. The Buyer's Actual Question

Special education buyers land on a CbD listing and ask: *"Will this work for my room, with my students, without me spending a weekend figuring it out?"*

The current descriptions answer "what the unit teaches" but not "what it looks like when you use it Monday." Every description body needs one concrete use-case sentence: "Open the unit, select the Lexile level that matches each student's reading access, and hand it out. The AAC vocabulary is already listed — no prep needed from the SLP before you can start."

### 2. The Three-Level Frame Is Your #1 Differentiator — Use It Harder

No other TPT seller in the special education nonfiction niche is offering three leveled versions with embedded AAC access AND standards alignment AND IEP goal stems in one purchase. That's extraordinary. But the current descriptions bury it.

Every description should state it explicitly within the first 3 sentences: **"One purchase. Three versions. Same standard, same expectation — every student accesses the content."**

That sentence is both your conversion hook and your differentiator. Put it right after the keyword-dense opening.

### 3. What's Included — Make It Scannable

TPT buyers skim. Every description needs a tight "What's Included" block (not buried in paragraph form) early in the description:

**What's included:**
- [X] leveled passages (V1/V2/V3 — three reading levels, same content)
- Teacher Guide with AAC Communication Partner guidance
- Vocabulary list: core and fringe words, symbol-ready
- Graphic organizers (one per version level)
- Rubric with IEP goal stems
- [Page count]

This format reduces buyer hesitation because it removes ambiguity about what they're paying for.

### 4. The Freebie Bridge

Every paid nonfiction listing should mention the Keiko Part 1 free preview in the description: "New to Communicate by Design? Start with the free Keiko preview — three leveled passages, no strings attached." This serves two functions: reduces purchase risk for first-time buyers, and drives traffic to the freebie listing (which drives Substack signups).

### 5. Bundle Callouts

Each listing should name its bundle explicitly with the savings amount. Buyers respond to specific numbers:
- Radium Girls + Frances Kelsey → "Bundle both as the Women's History Bundle and save $4."
- 504 Sit-In + Capitol Crawl → "Bundle both as the Disability Rights Bundle and save $3."

Place this in the final paragraph of every description in the relevant pairs.
