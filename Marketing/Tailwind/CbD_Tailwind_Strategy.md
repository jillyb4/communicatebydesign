## The Tailwind Step — Standard Workflow

**This step is required for every TPT product listing AND every Substack post.**
It is produced at the same time as the TPT listing package or post draft — not added later.

### For Every New TPT Product

When a new product is ready to list on TPT, the listing package must include:

#### Tailwind Pinterest Package (3 pins)
- **Pin 1** — Publish same day as TPT listing goes live
  - Title (≤100 chars): keyword-rich, lead with topic + "special education" + grade band
  - Description (150–200 chars): what it does + who it's for + 3–5 hashtags
  - Alt text: descriptive, no hashtags
  - Board(s): product-line board + "Communicate by Design Resources" brand board
  - Image direction: Canva graphic spec (navy background, product title, grade band, CbD wordmark, amber accent)
  - Link: direct TPT product URL
  - **3-image products (PB Companions, Nonfiction, Fiction):** Pin 1 = Image 1 (cover/pin graphic), Pin 2 = Image 2 (CommBoard/CAP), Pin 3 = Image 3 (Student Activities) — same TPT link, different visuals. See 3-Image Marketing Standard section below for full spec.

- **Pin 2** — Schedule 3–5 days after Pin 1
  - Visually distinct from Pin 1 (different headline angle, different layout)
  - Same link; different board order (brand board first, then product-line board)

- **Pin 3** — Schedule 7–10 days after Pin 1
  - Lead with a different hook (feature callout, audience callout, or seasonal angle)
  - Same link; pin to any relevant secondary board

#### Tailwind Instagram Package (1 carousel)
- Caption (125–150 chars + 5 hashtags): practical, ends with "Link in bio →"
- 3–4 slides: problem → product → who it's for → CTA
- Schedule: same day as TPT listing or +1 day

#### Tailwind Facebook Package (1 post)
- 2–3 sentences: what the product does + who it helps + direct TPT link
- Schedule: same day as TPT listing
- Note: Facebook reach is non-follower-heavy (99.6%) — write for a cold reader, not a follower

---

### For Every New Substack Post

When a Substack post is published, the distribution package includes:

#### Tailwind Pinterest Package (3 pins — educational content type)
- **Pin 1** — Schedule same day as publish (or up to 24 hours after)
  - Lead with the question the post answers (provocative, shareable)
  - Link: Substack post URL
  - Board: topic-relevant board (e.g., "Communication Partner Strategies" for ACL post) + brand board

- **Pin 2** — Schedule 3 days after Pin 1
  - Lead with the answer/takeaway (bold statement)
  - Same link; alternate boards

- **Pin 3** — Schedule 7 days after Pin 1
  - Lead with the evidence or research hook
  - Same link; rotate boards

#### Tailwind Instagram Package (1 carousel)
- Slide 1: The wrong assumption (bold, shareable hook)
- Slide 2: The one piece of evidence (plain language)
- Slide 3: The concrete takeaway (what to do Monday)
- Slide 4: "Full post at the link in bio — Communicate by Design"
- Schedule: same day as publish or +1 day

#### Tailwind Facebook Package (1 post)
- 2–3 sentences max
- **Schedule: 24–48 hours after Substack publishes** (subscribers see it first — respect that relationship)
- Plain text + Substack link preview (no image needed)
- After Tailwind posts to the page: manually drop the same link as a comment in the relevant SPED Facebook group thread

## Pinterest SmartLoop Rules

SmartLoop recirculates existing pins automatically. Rules for CbD:

- **Eligible for SmartLoop:** Any product pin older than 30 days that has ≥1 save
- **Not eligible:** Pins with seasonal hooks (Women's History Month pins should not loop in July)
- **Interval:** No pin recirculates to the same board more than once every 60 days
- **Board for loops:** "Communicate by Design Resources" brand board is the primary SmartLoop destination

### Getting the Canva "View" URL (required for Tailwind)

Every Canva image needs its own public share URL — one URL per product per image slot. Tailwind uses this as the image source.

**How to get the URL after a bulk import or single design export:**
1. Open the Canva design for that product
2. Click **Share** (top right)
3. Click **View link** (or "Copy link" — select View, not Edit)
4. Copy the URL (format: `https://canva.link/[shortcode]` or `https://www.canva.com/design/[ID]/view`)
5. Paste into the corresponding Airtable Products field: Image 1 → `fldYDDTNbdlE8AfNv`, Image 2 → `fldwAZkqxIEBReiNp`, Image 3 → `fldK2TRQ7xkodA4w8`

**One URL per product, per image.** If a template has multiple pages (e.g., nonfiction pin template has 7 pages), navigate to the specific page for that product before copying the view link — or export the single page and use the exported design's URL.

## Tailwind CSV — Generation Workflow

**Tailwind CSV is generated AFTER TPT listing.** The TPT product URL must be confirmed in Airtable before the CSV can be built. This is non-negotiable — the TPT link is the destination for all 3 pins.

### Required Sequence

```
1. Build product (docx + PDF)
2. QC pass
3. Upload to TPT → copy TPT product URL
4. Paste TPT URL into Airtable Products → [TPT URL] field (fldx9FesXwfqZhWYp is Pinterest Pin URL; TPT URL is fldbqjeqWjH6J5zRi)
5. Build 3 Canva images (bulk import for Image 1; bulk import for Images 2+3)
6. Export each Canva design → Share → View link
7. Paste Image 1 Share URL → Airtable fldYDDTNbdlE8AfNv
8. Paste Image 2 Share URL → Airtable fldwAZkqxIEBReiNp
9. Paste Image 3 Share URL → Airtable fldK2TRQ7xkodA4w8
10. Write Pin Titles, Descriptions, Alt Text → Airtable pin metadata fields
11. Generate Tailwind CSV from Airtable
12. Upload CSV to Tailwind → schedule 3 pins (Day 1 / Day 4 / Day 10)
13. After pins go live → paste Pinterest Pin URL → Airtable fldx9FesXwfqZhWYp (triggers daily brief auto-detect)
```

### Tailwind CSV Column Structure

The Tailwind CSV has one row per pin. A 3-image product = 3 rows (same TPT link, different image URL and pin copy).

| Column | Airtable Source Field | Notes |
|--------|-----------------------|-------|
| `pin_title` | `fldQV0iXyTZNZE1bq` (Pinterest Pin Title) | ≤100 chars. Use same title for all 3 or write 3 variants. |
| `pin_description` | `fldBoDY3YOfkBeIfl` (Pinterest Pin Description) | 150–200 chars + 3–5 hashtags. Write 3 variants — one per image angle. |
| `alt_text` | `fldQHiO3oo2D10n0a` (Pinterest Alt Text) | Descriptive, no hashtags. |
| `destination_url` | `fldbqjeqWjH6J5zRi` (TPT URL) | The TPT product URL — same on all 3 rows. |
| `image_url` | `fldYDDTNbdlE8AfNv` / `fldwAZkqxIEBReiNp` / `fldK2TRQ7xkodA4w8` | One per row: Image 1 URL, Image 2 URL, Image 3 URL |
| `board` | (manual) | Pinterest board name. Row 1: product-line board + brand board. Row 2: brand board first. Row 3: secondary board. |
| `schedule_date` | (manual) | Row 1 = TPT listing day. Row 2 = +4 days. Row 3 = +10 days. |

**Pin description variants by image angle:**
- **Pin 1 (Cover):** Lead with what the product is + who it's for
- **Pin 2 (CommBoard / CAP):** Lead with the communication access angle ("Every student has a way to respond")
- **Pin 3 (Student Activities):** Lead with the standards/evidence angle ("Grade-level ELA — designed for the whole class")
