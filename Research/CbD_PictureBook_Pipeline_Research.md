*Created: 2026-04-04 | Product Line: Picture Book Companions (K–3)*
*Purpose: Title selection framework + candidate list for 25–30 book pipeline.*

This document supplements `Picture_Book_Companions_Research_Foundation.md`. That file grounds the instructional design. This file answers the question: **which books belong in this line, in what order, and why.**

## 1. Selection Framework (Non-Negotiable Criteria)

Every title in the CbD Picture Book Companions pipeline must pass all four gates before entering the build queue.

### Gate 1 — Core Vocabulary Richness
The book's text and storyline must support authentic, recurring use of **Foundation Core vocabulary** (Growing Core deck, 170 words). The companion is only as strong as the core vocabulary it can model throughout a complete shared reading. A book with minimal verb/preposition/descriptor opportunity = weak ALS platform.

**Target:** 8–12 core words that appear naturally in the book's context and can be modeled on any AAC system throughout all three reads. These are not bolted on — they come from the story itself.

**Fitzgerald Key distribution goal per companion:**
- Green (verbs/actions): 3–5 core words (go, want, help, make, feel, change, stop, can, need)
- Orange (descriptions): 1–2 (big, little, different, same, brave)
- Blue (little words/connectives): 1–2 (more, all, not, that, on, in)
- Yellow (people/pronouns): 1 (she, he, they, her, his)
- Pink (social): 1 (yes, no, thank you, please, stop)
- White (nouns/other): 2–3 unit-specific fringe nouns only

**Note:** Fringe vocabulary (book-specific nouns like "wheelchair," "Capitol," "water") is intentionally limited to 8–12 words per companion. The core-to-fringe ratio must stay weighted toward core — core language is what transfers across settings. Fringe is a hook, not the meal.

### Gate 2 — K–2 ELA Standard Alignment
Each companion targets **one or two anchor RL or RI standards** appropriate for K–2. The skill focus is what the teacher teaches. The AAC access layer is how every student participates. The standard does not change based on access method.

**Priority standards for picture book companions (Common Core K–2):**

| Standard | Focus | Notes |
|---|---|---|
| RL.K.3 / RL.1.3 / RL.2.3 | Characters: feelings, responses, changes | Best fit for character-centered books |
| RL.K.1 / RL.1.1 / RL.2.1 | Key details: who, what, where, when, why | Dialogic Reading CROWD prompts built for this |
| RL.K.6 / RL.1.6 / RL.2.6 | Point of view: author's purpose, character perspective | Advocacy and identity books |
| RL.K.2 / RL.1.2 / RL.2.2 | Retell: key details + central message/lesson | Strong for repetitive-text books |
| RI.K.3 / RI.1.3 / RI.2.3 | Connection: events, ideas, concepts (informational) | Nonfiction picture books (biography, advocacy) |
| RI.K.6 / RI.1.6 / RI.2.6 | Author's purpose (informational) | Civil rights / disability history titles |

Each companion specifies grade-level target (K, 1, 2, or K–2) and the primary + optional secondary standard. The standard applies to ALL students. The scaffold is the access method — never the expectation.

### Gate 3 — EBP + HLP + CEC/IRIS Instructional Activities
Every companion must include **at least 3 whole-class instructional activities** that are:
- Grounded in at least one HLP (High Leverage Practice, CEC 2017)
- Validated by IRIS Center or CEC literature
- Designed for whole-class use — not pull-out, not specialist-only
- Accessible to complex communicators without modification (access method varies; task does not)

**Evidence-Based Practices + HLPs mapped to K–2 shared reading:**

| EBP / Activity | HLP | IRIS/CEC Source | What it does in a companion |
|---|---|---|---|
| **Aided Language Stimulation (ALS)** | HLP 15 (Scaffolded supports) | IRIS: AAC Implementation Module | Partner models core + fringe on AAC system while speaking — throughout every read |
| **Interactive Read-Aloud (IRA)** | HLP 16 (Explicit instruction) | IRIS: Reading Comprehension | Structured read with planned stops; teacher holds text; all students respond |
| **Dialogic Reading (PEER/CROWD)** | HLP 14 (Flexible grouping) | Whitehurst et al. 1988; IRIS Vocab module | Shifts students from listeners to storytellers across 3 reads |
| **Explicit Vocabulary Instruction** | HLP 12 (Systematic instruction) | Beck, McKeown & Kucan 2002; IRIS Vocab | Pre-teach 3–5 fringe words before read; revisit during; apply after |
| **Predictable Chart Writing / Write-Ables** | HLP 20 (Social skills — adapted) | Hanser & Erickson 2007; CEC Literacy | Whole-class sentence frame; students contribute via any access method |
| **Response Opportunities (choral, partner, symbol)** | HLP 18 (Strategies for active engagement) | IRIS: Intensifying Instruction | Every stopping point includes 3 response pathways: verbal, symbol/board, gestural |
| **Graphic Organizers / Story Maps** | HLP 15 (Scaffolded supports) | IRIS: Learning Strategies | Visual organization of character/event/setting — usable with symbol-based response |
| **Repeated Reading (3-read structure)** | HLP 16 (Explicit instruction) | NICHD 2000; NRP 2000 | Read 1 = exposure / Read 2 = comprehension / Read 3 = skill application |
| **Systematic Prompt Hierarchy (MODE 1 only)** | HLP 22 (Structured feedback) | IRIS: Prompting | 5-level prompt hierarchy applied ONLY in Mode 1 instructional contexts; not default |
| **Symbol-Supported Response** | HLP 15 | ISAAC; ASHA evidence maps | Core symbol cards + communication board available for every response opportunity |

**IRIS Center modules directly applicable to this line:**
- "Providing Instructional Supports: Facilitating Mastery for All Students" — HLP 15
- "Vocabulary Instruction" — Beck & McKeown connection
- "Comprehension Instruction K–3" — IRA alignment
- "AAC Implementation in the Classroom" — ALS + partner modes
- "Intensifying Instruction" — response opportunities + prompt fading

**CEC High Leverage Practices applied per companion (minimum):**
- HLP 12 (Systematic instruction) — vocabulary sequence
- HLP 15 (Scaffolded supports) — ALS + communication board
- HLP 16 (Explicit instruction) — IRA structure with planned stops
- HLP 18 (Flexible engagement) — 3 response pathways per activity

### Gate 4 — Whole-Class + Complex Communicator Design
The companion is designed for **the general education classroom first.** Every activity works for every student. The AAC access layer is not a separate track — it is the design.

Design principle: *"The scaffold varies, the expectation does not."*

- All activities include: verbal response pathway + symbol/board response pathway + gestural/physical response pathway
- Partner scripts are written for Circle 3 competence (paraprofessional with no AAC training)
- No activity requires an SLP to implement — the companion is the professional development
- Communication boards are print-and-laminate ready, no device required
- Symbol cards are pre-made — no teacher prep beyond printing

**9. This is How We Talk — Jessica Slice / Parwinder Singh**
- Grade: K–1 | Standard: RL.K.3 / RL.K.2 (retell + central message: all communication is valid)
- Status: Pipeline candidate — needs full gate review
- Core vocab anchors: talk, feel, can, all, same, different, more, help, want, go, together, my
- Fringe vocab: sign, stim, clap, device, write, gesture, smile, quiet, loud, community
- Why it fits: Jessica Slice is a disabled author. The book explicitly and joyfully names every communication method — speaking, signing, writing, stimming, clapping, using devices — as equally valid. - TPT competition: Very low — recent publication, disability-authored, no AAC companion exists
- Priority: **5** — brand thesis made visible; whole-class inclusion message; exceptional ALS platform

## 4. Vocabulary Architecture for This Line

Every companion uses this structure. No exceptions.

**Foundation Core Words (modeled throughout ALL reads via ALS):**
The same ~12 core words recur across all companions — these are the Growing Core backbone.
go · want · help · feel · can · change · more · all · different · same · try · make

**Unit Core Words (specific to this book's narrative — 2–4 additional core words):**
Selected from the book's natural language. Must appear in the text.

**Fringe Words (book-specific — max 10, pre-teach before Read 1):**
Concrete nouns/verbs tied to the book's content. Every fringe word has an ARASAAC symbol. SLP/AAC team must add to system before instruction.

**Communication Board layout per companion:**
- 4 col × 5 row
- Foundation core in FK colors (green verbs, blue little words, orange descriptors, pink social)
- Book-specific fringe in bottom rows with FK border
- Print-and-laminate ready, 8.5×11

## 5. Three-Read Structure (Locked for All Companions)

**Read 1 — Exposure + Vocabulary Introduction**
- Teacher reads aloud; students listen and attend
- ALS throughout: partner models core vocabulary on AAC system while teacher reads
- Planned stops: 3–4 pre-scripted vocabulary stops (fringe words in context)
- Response: low-demand (gesture, gaze, symbol tap — any access method)

**Read 2 — Comprehension + Dialogic Reading**
- PEER sequence at planned stopping points
- CROWD prompts: Completion + Recall + Open-ended
- Response: symbol board or verbal; Write-Able sentence frame introduced
- Graphic organizer introduced (character map, story sequence, or problem/solution)

**Read 3 — Skill Application**
- ELA standard is the target (character feelings, retell, author's purpose, etc.)
- Whole-class activity: Predictable Chart Writing OR symbol sort OR story map
- Every student contributes via their access method
- ALS continues throughout
