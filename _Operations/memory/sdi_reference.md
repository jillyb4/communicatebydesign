## SDI vs. Accommodation — Quick Distinction

| | Accommodation | Specially Designed Instruction |
|--|---------------|-------------------------------|
| Changes | Format or setting | Methodology or content delivery |
| Standard | Unchanged | Unchanged (access pathway adapted) |
| Who provides | Any staff | Qualified special educator |
| IEP location | Accommodations section | SDI/Services section |
| CbD example | Extended time | V3 adapted text + partner read-aloud protocol |

**The test:** Does this change *how the instruction is taught*, or only *how the student accesses or demonstrates* the same instruction? If the instruction itself changed → SDI. If only the format changed → accommodation.

## Which CbD Components Are SDI-Level

**Must connect to an IEP goal (SDI):**
- V2 or V3 adapted text (content + methodology adaptation)
- Vocabulary Pre-Teaching via Vocabulary Preview Routine (explicit instruction methodology)
- Communication Access Packet used with SLP collaboration (delivery adaptation)
- AAC Response Pathway as the primary response modality (delivery adaptation)
- Partner Read-Aloud Protocol documented as the instructional delivery mode
- V3 Vocabulary Instruction step using Descriptive Teaching Model (methodology)

**Accommodation level (IEP accommodations section):**
- Symbol Cards PDF (visual support)
- Extended time
- Read-aloud as a test accommodation
- Sentence frames as a test accommodation
- Preferential seating/grouping

## SDI Documentation Trail (Minimum Required)

When a CbD unit is used as SDI:
1. **IEP identifies** the disability-related need this SDI addresses
2. **IEP annual goal** reflects the unit's standard with measurable criterion
3. **IEP SDI section** names the specific adaptation (which version + which supports)
4. **CAP delivery** to SLP documented (timestamp = implementation evidence)
5. **Session Tracker** / rubric data collected at each unit activity
