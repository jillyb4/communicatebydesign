## Adding New Research
1. Drop the article or document in the Research/ folder if it has a file
2. Add a record to the Airtable Research Library (tblKpzHsm4HHe3qsa)
3. Fill: Title, Authors, Year, Journal/Source, Topic Area, Key Findings, CbD Application, CbD File path, Full Text Access
4. The weekly `cbd-research-scan` task adds journal articles automatically — review on Mondays
