# CbD Differentiation — Operational Reference
*Added: 2026-03-29 | Full research: `Research/CbD_Differentiation_Framework.md`*

**Load this file when:** writing or reviewing any nonfiction passage or activity, creating a new text-based product line, or QC-checking V1/V2/V3 version consistency.

---

## The Locked Constants (Never Change Across Versions)

| Element | Rule |
|---------|------|
| ELA standard | Identical across V1/V2/V3 |
| Essential question | Identical across V1/V2/V3 |
| Vocabulary list | Same words, same Top 5, same CAP |
| Historical events and evidence | Every fact in V1 must appear in V3 |
| Skill type | Close Reading / Text Structure / CER / Author's Purpose / Sourcing — does not change by version |
| Response task category | If V1 = claim + evidence, V3 = claim + evidence (format scaffolds, task does not) |

---

## The Three Differentiation Mechanisms

### 1. Text Complexity (Lexile)

| Version | Lexile | Words/Passage | Design Intent |
|---------|--------|--------------|---------------|
| V1 | 900–1050L | 550–650 | Near-original source language |
| V2 | 650–800L | 450–550 | Restructured sentences; same events |
| V3 | 400–550L | 330–400 | Chunked sentences; embedded vocabulary access |

**How to reduce complexity (in order):**

1. **Sentence decomposition** — break long multi-clause sentences. Preserve all information.
2. **Syntactic simplification** — passive → active; embedded clauses → separate sentences; abstract noun phrases → concrete descriptions.
3. **Knowledge demand reduction** — embed Key Vocabulary boxes in V3 only (max 3–4 terms; student-facing definitions; one sentence each).

**Hard rule:** V3 does NOT remove evidence. Every fact, example, and piece of evidence present in V1 must appear in V3.

---

### 2. Activity Scaffold Gradient

| Format | Version | Pattern |
|--------|---------|---------|
| Open-ended | V1 | Blank space; student produces full response |
| Partial structure | V2 | Sentence frame; student fills evidence and analysis |
| Choice/cloze access | V3 | Options embedded; student selects AND supplies at least one content element |

**The access test:** Can the V3 activity be completed without engaging with the text? If yes → the scaffold has reduced the expectation. Revise.

**Sentence frames rule:** Present in all versions. May simplify in language complexity V1→V3. Must:
- Require student to supply content (not just copy)
- Target same grammatical structure (claim + evidence, cause + effect, etc.)
- Use shared response core vocabulary: `because, show, prove, agree, different, same, not, true, wrong`

---

### 3. AAC Response Pathway (Overlay — All Versions)

- Core response vocabulary on communication board across ALL activities
- Fringe vocabulary pre-programmed via CAP → SLP (1–2 week lead time)
- All prompts written to be read aloud
- All response options are speakable (AAC output, oral, or written — all valid)
- Version is a READING access decision — not an AAC use decision. An SGD user can be on V1.

---

## Version Assignment — Quick Protocol

1. **Reading level** → V1 (900–1050L range), V2 (650–800L range), V3 (below 500L / partner read-aloud primary)
2. **IEP goal** → Grade-level standard with accommodations = V1/V2; Modified content = V2/V3
3. **Response modality** → Does not determine version. Activates the AAC pathway within the version.
4. **FAPE** → Version assignment is an IEP team decision. Document it.

Version assignment is not permanent. Movement from V3 → V2 during the year = progress. Plan for it.

## Common Differentiation Errors

| Error | Rule Violated |
|-------|--------------|
| Removing an event from V3 to shorten the passage | Locked constants — all facts must appear in V3 |
| V3 activity asks for identification only (not analysis) | Activity scaffold must maintain the response task category |
| Different vocabulary lists across versions | Vocabulary is locked — same list for all versions |
| V3 sentence frame supplies the evidence | Frame must require student to supply content |
| V3 embedded vocab box adds new info not in any passage | Embedded support = vocabulary only; no new factual content |
| Version assigned by disability label, not reading access need | Version = reading access decision; SGD users can be V1 |

## QC Gates (Run Before Finalizing Any Version)

**Gate 1: Text Complexity**
- [ ] V1/V2/V3 Lexile confirmed within bands?
- [ ] Word count within target for each version?
- [ ] All V1 facts present in V3?

**Gate 2: Activity Scaffolds**
- [ ] All versions address same skill type?
- [ ] V3 choice/cloze still requires student to supply content?
- [ ] Sentence frames present in all versions?
- [ ] Response task category identical across versions?

**Gate 3: AAC Pathway**
- [ ] Vocabulary list identical across versions?
- [ ] Key Vocabulary boxes in V3 only?
- [ ] All prompts readable aloud?
- [ ] Core response vocabulary in all activity scaffolds?
