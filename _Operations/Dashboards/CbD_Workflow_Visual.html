Critical Rule / Non-Negotiable
Phase 0 · Concept
0a
Answer Core Questions
What is the instructional gap? What ELA standard + skill? Who is the buyer? Is there a meaningful competitive gap? Document concept validation.
0b
Research Topic
Primary sources, historical context, photos, first-person narratives. Assess material richness, whose voices are present/missing, existing curriculum coverage.
0c
Define Scope
Age range (grade band) and differentiation strategy. Confirm V1/V2/V3 Lexile targets. Same skill + same expectation + different access model.
0d
Passage Architecture
Decide passage count (1–4): 1 = mini-lesson/freebie · 2 = comparison/shift · 3–4 = full narrative arc. Confirm pricing tier.

Phase 3 · QC
1
Content & Standards
All 22 sections present. Vocabulary table correct. Skill aligned to standard. Grade band appropriate. Strengths-based language throughout.
2
AAC + Accessibility
WCAG AA on all docs. No oral-response-only activities. Core/Fringe words have symbols. CAP is present + usable. Rubric behavioral.
3
Design & Brand
Header/footer consistent. Product colors correct (#006DA0 teal docs, not #00B4D8). Fonts correct. No typos.
4
PDF Conversion
Nonfiction/Fiction/Poetry: Word → Save As PDF (not LibreOffice). Picture Books: LibreOffice headless only. Verify fonts + layout preserved.
Phase 4 · Launch
1
TPT Listing
Output 0 first (title, description, keywords). All files assembled in _TPT/ folder. Preview PDF included. Live product URL confirmed.
2
Canva Cover
Use build_covers_v2.py. Render PNG 1200×1600. Update Pinterest pin template if needed. Store in Pending folder.
3
Airtable + Sync
Products table: mark Docx Built ✓ + QC Passed ✓. Run sync_vocab_to_airtable.js at BUILD TIME (not at TPT listing). Launch Pipeline: update status.
⚠ Critical Rules (Never Violate)
Non-Negotiable Requirements
● Vocabulary sync runs at BUILD TIME, not TPT listing. Never sync stubs.
● Every unit has all 22 standard sections. No exceptions.
● PDF: Word → Save As → PDF for nonfiction/fiction/poetry. Never LibreOffice for those lines.
● AAC core/fringe word table required — makeTable(), not bullets.
● Use #006DA0 teal in docs, not #00B4D8. WCAG AA always.
● No oral-response-only activities. Every response has a non-speech pathway.
● Symbol pages for picture books: Python/ReportLab ONLY, never inside .js.
● Completed tasks cleared from dashboards immediately. Never show stale statuses.
